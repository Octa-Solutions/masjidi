This repository contains the **core module** used in [tv.masjidi.app](https://tv.masjidi.app) providing essential logic and utilities powering it.

## Guidelines

- **Sensitive Module:** This package is the core logic for tv.masjidi.app and is considered sensitive.
- **No Feature Contributions:** We are **not currently accepting feature contributions** to this repository, as changes may impact production systems for tv.masjidi.app.
- **Bug Reports:** You may open issues for bugs or security concerns, but feature requests and PRs adding new features will not be accepted at this time.

## Packages

This monorepo contains:

- `@masjidi/core` &mdash; Core logic and utilities for tv.masjidi.app
- `@masjidi/common` &mdash; Shared utilities and types
